`id` 시작하는 애의 메모리 주소를 말함.

많이 쓰는 정수들은 이미 메모리 주소를 갖고있다. 따라서, 1004가 아닌 10일 때는 같은 주소가 나옴

```python
x = 1004
y = 1004
a = b = 1004
print(a, b, x, y)

print(id(a), id(b), id(x), id(y), id(1004))
```

그 외의 수는 다 다른 값이 나옴. a=b=1004때문에 a랑 b만 같다.
